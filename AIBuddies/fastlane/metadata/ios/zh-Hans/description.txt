AI Buddies 是面向 Claude Code 与 Codex 重度用户的用量监控与优化教练。它读取本地命令行写下的用量数据，统一展示 token、等效费用、5 小时与每周额度窗口，并按项目和模型拆分，给出"省钱、防限流、防浪费"的可执行建议。

• 跨端统一：在一处看清 Claude 与 Codex 两端用量
• 额度窗口：Codex 权威剩余% + 重置倒计时；Claude 用量估算代理（明确标注）
• 等效费用：按 token 估算"相当于花了多少钱"，回答订阅值不值
• 优化教练：贵模型用在小任务、缓存命中偏低、临近上限等，给出具体建议
• 隐私优先：数据默认留在本机；iOS 通过你的私有 iCloud 同步，绝不上传代码或对话
• macOS 菜单栏 + 仪表盘；iOS App + 主屏/锁屏小组件；临近上限本地提醒

注意：AI Buddies 不会登录你的 AI 账号，仅读取官方命令行在本地写下的用量产物。
